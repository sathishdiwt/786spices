📱 WHATSAPP SHARING GUIDE
═══════════════════════════════════════════════════

You have 2 options to send site reports to WhatsApp:

───────────────────────────────────────────────────
OPTION 1: HTML Tool (RECOMMENDED) ⭐
───────────────────────────────────────────────────
File: sitereport_whatsapp.html

HOW TO USE:
1. Open sitereport_whatsapp.html in any web browser
2. Fill in the report details:
   - Report Date (auto-fills with today)
   - Site Name
   - Progress % for each location
3. Click "📱 Export for WhatsApp" button
4. Choose between:

   A) 📋 COPY TEXT
      • Shows formatted text report
      • Click "Copy to Clipboard"
      • Open WhatsApp → Paste message
      ✅ Best for: Quick text updates

   B) 📸 DOWNLOAD IMAGE
      • Generates a professional screenshot
      • Downloads as PNG image
      • Open WhatsApp → Attach image
      ✅ Best for: Polished, shareable reports

5. Alternative: Use "🖨️ Print Report" to save as PDF


───────────────────────────────────────────────────
OPTION 2: Text Template (MANUAL) 
───────────────────────────────────────────────────
File: whatsapp_text_template.txt

HOW TO USE:
1. Open whatsapp_text_template.txt in any text editor
2. Fill in:
   - [DATE] with today's date
   - [SITE NAME] with your site name
   - Replace __% with actual progress numbers
   - Add notes at the end
3. Copy all text (Ctrl+A → Ctrl+C)
4. Open WhatsApp → New message
5. Paste (Ctrl+V)
6. Send!

✅ Best for: No frills, quick manual updates
